# Save my S

A Chrome extension that redacts sensitive info — emails, API keys, credit cards, and more — before it's sent to AI chat UIs.

**100% local. No data ever leaves your browser.**

Works on 20 AI chat sites including ChatGPT, Claude, Gemini, Copilot, Perplexity, Mistral Le Chat, Grok, DeepSeek, and more.
